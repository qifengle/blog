## Lipper的个人博客

### 使用 React + Redux + React Router + webpack 搭建个人博客

### 技术栈
#### react + redux + react-router + ES6/7 + webpack + express

### 演示地址
[Lipper的个人博客](https://119.29.38.170)


## Usage

### 安装
```
git clone https://github.com/qifengle/blog.git

cd react-projects/blog

npm install
```

#### 开发模式：
```
npm run dev
```

#### 产品模式：
```
npm run build
```
